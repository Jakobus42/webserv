\doxysubsection{Detailed Description}
Abstract base class for logging functionality. 

{\ttfamily \mbox{\hyperlink{classlogger_1_1ALogger}{ALogger}}} provides a framework for logging systems, managing log levels (DEBUG, INFO, WARNING, ERROR, FATAL) and formatting log messages with color-\/coding. It defines\+:


\begin{DoxyItemize}
\item {\ttfamily \mbox{\hyperlink{classlogger_1_1ALogger_a601ae64c4455599cb8be6e03edd3f469}{set\+Level()}}} and {\ttfamily \mbox{\hyperlink{classlogger_1_1ALogger_a0ff4e250a3f838c5f766901aac53e645}{get\+Level()}}}\+: To manage log levels.
\item {\ttfamily log()}\+: To log messages (to be implemented by derived classes).
\item {\ttfamily get\+Log\+Level\+As\+String()}\+: Returns a string representation of the log level, with colors.
\end{DoxyItemize}

Derived classes must implement the logging logic.
