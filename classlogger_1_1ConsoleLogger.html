Detailed Description

Logs messages to the console.

The ConsoleLogger class is a concrete implementation of the ALogger abstract class, designed to output log messages to the standard output (console). It supports different log levels and formats log messages accordingly.

This logger is suitable for scenarios where you want to see log messages in the console for debugging or monitoring purposes. It inherits from ALogger, which provides the foundational logging functionality.
