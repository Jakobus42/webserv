This function converts a hexadecimal string to an unsigned integer. The function will stop converting when it encounters a character that is not a hexadecimal digit. If the result would exceed the maximum value of a 32-bit unsigned integer, the function returns 0 and sets the ret parameter to -1.

Parameters

retA reference to an integer that will be set to -1 in case of incorrect input.
Returns
