\doxysubsection{Detailed Description}
Abstract base class for logging functionality. 

{\ttfamily \mbox{\hyperlink{classlogger_1_1ALogger}{A\+Logger}}} provides a framework for logging systems, managing log levels (D\+E\+B\+UG, I\+N\+FO, W\+A\+R\+N\+I\+NG, E\+R\+R\+OR, F\+A\+T\+AL) and formatting log messages with color-\/coding. It defines\+:


\begin{DoxyItemize}
\item {\ttfamily \mbox{\hyperlink{classlogger_1_1ALogger_a601ae64c4455599cb8be6e03edd3f469}{set\+Level()}}} and {\ttfamily \mbox{\hyperlink{classlogger_1_1ALogger_a0ff4e250a3f838c5f766901aac53e645}{get\+Level()}}}\+: To manage log levels.
\item {\ttfamily \mbox{\hyperlink{classlogger_1_1ALogger_a432d93d4ad4a568c178492e9912e145a}{log()}}}\+: To log messages (to be implemented by derived classes).
\item {\ttfamily \mbox{\hyperlink{classlogger_1_1ALogger_aeec7dc86db5a5df42d02b494e24802cf}{get\+Log\+Level\+As\+String()}}}\+: Returns a string representation of the log level, with colors.
\end{DoxyItemize}

Derived classes must implement the logging logic.
