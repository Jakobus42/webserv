\doxysubsection{Detailed Description}
Logs messages to the console. 

The {\ttfamily \mbox{\hyperlink{classlogger_1_1ConsoleLogger}{Console\+Logger}}} class is a concrete implementation of the {\ttfamily \mbox{\hyperlink{classlogger_1_1ALogger}{A\+Logger}}} abstract class, designed to output log messages to the standard output (console). It supports different log levels and formats log messages accordingly.

This logger is suitable for scenarios where you want to see log messages in the console for debugging or monitoring purposes. It inherits from {\ttfamily \mbox{\hyperlink{classlogger_1_1ALogger}{A\+Logger}}}, which provides the foundational logging functionality.
