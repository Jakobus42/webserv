Detailed Description

Abstract base class for logging functionality.

ALogger provides a framework for logging systems, managing log levels (DEBUG, INFO, WARNING, ERROR, FATAL) and formatting log messages with color-coding. It defines:

  • setLevel() and getLevel(): To manage log levels.
  • log(): To log messages (to be implemented by derived classes).
  • getLogLevelAsString(): Returns a string representation of the log level, with colors.

Derived classes must implement the logging logic.
